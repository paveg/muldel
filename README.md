# Multi Delivery Aggregation

This application is delivery service for more convenience.

## Introduction 

* Ruby version: 2.6.3

* Technical Structure

    * Rails: 5.2.3
    * Ruby: 2.6.3
    * React.js
    * TypeScript

* Configuration

* Database creation

 ```bash
$ bundle exec rake db:create
```

or 

```bash
$ docker-compose up
```

* Database initialization

```bash
$ bundle exec rake db:migrate
```

* How to run the test suite

```bash
$ bin/rspec spec
```
